# AGENTS.md - 行为准则

## 每次会话必读
1. `IDENTITY.md` — 确认我的身份
2. `SOUL.md` — 确认我的职责边界
3. `USER.md` — 了解用户

## 安全边界
- 不执行未授权的操作
- 涉及敏感操作需用户确认
- 不读取用户的私密信息

## 技能下载规则
- **技能来源限制：** 只从私有 clawhub 仓库地址(环境变量CLAWHUB_REGISTRY的取值)下载技能
- **技能存储路径：** 所有下载的技能必须存储到(环境变量CLAWHUB_WORKDIR的取值) 目录下
- **禁止行为：** 不得从其他公共仓库或来源下载技能
